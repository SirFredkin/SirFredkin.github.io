流体力学

Created2022-10-06|Updated2022-10-21
|Post View:

前言

终于开始写力学的内容力！有哪里写错了记得叫我改（bushi）。这里只写了考试的知识点，要说看了我这个之后，还是不可能从头学懂流体力学的。

流体静力学

牛顿内摩擦定律

这个按理说不该出现在这里，就是简单介绍下，后面对于“粘度”这个东西肯定是还要讲讲的。

牛顿内摩擦定律只适用于牛顿流体(Newtonian Fluid)，即：剪应变和速度的梯度成正比的流体。这也是内摩擦定律的主要公式：

$$ \tau = \mu \frac{\partial u}{\partial y}$$

牛顿内摩擦定律指出：流体无法承受剪应力，流体的粘度只会影响流动的速度，无法使流动停下来。

压力相关

对于一个点的压强，有：

$$ p_x = p_y = p_z = p$$

即：x,y,z 各个方向的压强相等。

流体平衡的微分方程（欧拉平衡方程）：

为了大家读着方便懂，我把方程写成最简单的形式，在xyz三个方向的方程：

$$ f_x - \frac{1}{\rho} \frac{\partial p}{\partial x} = 0$$
$$ f_y - \frac{1}{\rho} \frac{\partial p}{\partial y} = 0$$
$$ f_z - \frac{1}{\rho} \frac{\partial p}{\partial z} = 0$$

其中$\rho$ 是密度，$f_x$这些是体积力（例如g）

静止状态下，压力微元的大小：

$$dp = \rho (f_x dx+f_y dy+ f_z dz)$$

对这个$dp$进行积分，即可得到压力在这个空间的分布。

基本上这样做就可以知道压力分布，从而解出一系列问题了。不过最好还是了解一下比如说一些特殊情况下的解法：

均匀加速状态下的压力分布：

如图的情况，流体受重力，同时小车有一个水平的加速度a。可以写出dp此时的方程：

$$dp = \rho(-adx-gdz)$$

这样就求出了p的分布。注意积分后会出现常数，这里需要用自由液体表面的边界条件来解决，即与空气接触的点压强为大气压。

还有一个很有用的是：液体形成了一个倾角$\alpha$,存在公式：

$$\tan \alpha = \frac{a_x}{g}$$

匀速转动的容器

这样的情况，受力情况为：

$f_r = \omega^2 r , f_z = -g$

$$dp = \rho(r\omega^2 dr - gdz)$$

下面是压强的分布：

$$p = \rho g (\frac{r^2\omega^2}{2g}-z)+C_3$$

等压面的方程：

$$z = \frac{1}{2g}r^2\omega^2 + C_4$$

静水压大小计算

平面情况

$$P = p_0 A + \gamma h_c A$$

其中hc为形心所在的高度（距水面垂直高度），$\gamma$为比重。

还有一个重要的，是压力作为一个等效的集中力作用在哪。

比如一个斜着放的平面,

$$L_d = L_c + \frac{Ic}{L_c A}$$

复习下一些常见的图形的$I_c$:

长方形，有$I_c = \frac{BL^3}{12}$

圆形，有$I_c = \frac{\pi R^4}{4}$

曲面情况

通常对于曲面的静水压，需要分成两个分量来考虑：x向和z向（竖直方向）

水平方向：
$$F_x = \rho g h A_x$$
其中$A_x$事曲面在xy面上的投影,h事曲面投影过的中点的水深。

竖直方向：
$$F_z = \rho g h V$$

其中V事曲面“上面”的水的体积。

流体动力学

首先我们需要对于流动进行分类：

理想流体和粘性流体：粘度$\mu$是否为0

可压缩流体和不可压缩流体：$\rho = or \neq const$

稳定和不稳定：$\frac{\partial}{\partial t} = or \neq 0$

均匀流动和不均匀流动：

流动的维度：1维、2维、3维

质点导数

已知流体的速度分布，但要求流体加速度，可不只是对时间求个导。

比如对于x方向，需要做的是对速度进行全微分，之后的细节我就不写了：

$$a_x = \frac{\partial u}{\partial t}+ u \frac{\partial u}{\partial x}+v\frac{\partial u}{\partial y}+w \frac{\partial u}{\partial z}$$

也就是说，先求t的偏导数，再u，v，w乘上u对于x，y，z的偏导数。

迹线

描述的是流场中一个质点的运动轨迹

求法：已知$v_x$和$v_y$的方程，解微分方程，之后得到x，y关于t的参数方程，消去t即可。

流线

描述的是某一时刻，许多质点的速度矢量。

流线方程：

$$\frac{dx}{v_x} = \frac{dy}{v_y}$$

连续性方程（重要）

假设对于一个管流，流体：

1.不能从管壁流出去

2.流体是连续介质

3.质量不可能变成能量

鉴于在本课程中，密度是常数，经常有这样的条件：

$$V_1 A_1 = V_2 A_2$$

伯努利方程

做过很多简化(稳定、不可压缩、理想、沿着流线流动、只有重力)之后，最基本的形式：

$$z_1+ \frac{p_1}{\rho g}+\frac{V_1^2}{2g} = z_2+ \frac{p_2}{\rho g}+\frac{V_2^2}{2g}$$

经常在两个平面之间列写。

伯努利方程的应用1：曲率很小：$r \rightarrow \inf$

$$z_1+\frac{p_1}{\rho g} = z_2 + \frac{p_2}{\rho g}$$

伯努利方程的应用2：在整个流场机械能守恒的话：

$$Vr = C_1$$

动量方程

这个方程揭示了合外力大小和在该方向上流体速度改变的量。

比如在x方向上有：

$$\rho Q(u_2 - u_1) = \Sigma F_x$$

其中Q是流量。

做这方面的题的时候要注意，作为流体，静水压给墙壁有压力，自身就会收到反作用力。需要考虑进去。

动量矩方程

和动量方程形式很相似，如果说不考虑流量对于面积的变化，则形式较为简单：

$$\rho Q_1 (\vec{r_1} \times \vec{V_1}) - \rho Q_2 (\vec{r_2} \times \vec{V_2}) = \Sigma (\vec{r} \times \vec{F})$$

伯努利方程的补充：

雷诺数 Re

$$Re = \frac{\rho Vd}{\mu}$$

其中d是水力直径，与截面形状有关。

$$d = \frac{4A}{\chi}$$

其中A是面积，$\chi$是湿周，指固体被流体浸湿的周长。例如：圆管是$\chi = \pi d$，长方形是$\chi = 2*(a+b)$，套管是：$\chi = \pi d_1+\pi d_2$。水中有很多管是$\chi = S_1 S_2 - \pi d^2/4$

雷诺数和伯努利方程的联系在于，层流状态下Loss Factor的变化：

$$\lambda = \frac{64}{Re}$$

湍流的情况下，$\lambda$ 是Re的函数。

沿程能量损失

这是一个修正项，因为流动的方向上一直存在着切应力阻碍流动。

$$h_f = \lambda \frac{l}{d}\frac{V^2}{2g}$$

$\lambda$的值在上面讨论了怎么求。

局部能量损失

由于流动的时候比如截面大小突然变化，在这个交界处出现了能量损失。

$$h_j = K\frac{V^2}{2g}$$

其中K为一个系数。下面将对它进行讨论

对于局部能量损失，K的取值

众所周知，管道直径的突然膨胀、收缩都会导致能量的损失。

  1. 管道突然变大

$$K_1 = (1-\frac{A_1}{A_2})^2$$

对于慢慢变大，直接有经验公式来讨论$h_j$:

$$h_j = \phi \frac{(V_1-V_2)^2}{2g}$$

  1. 管道突然变小

$$K_2 = 0.5(1-\frac{A_2}{A_1})^2$$

加入这俩之后的伯努利方程：

$$z_1+\frac{p_1}{\rho g}+\alpha \frac{V_1^2}{2g} = z_2+\frac{p_2}{\rho g}+\alpha \frac{V_2^2}{2g}+ h_w + E_m$$

其中$H_w$是能量损失,$E_m$是外界输入/输出的能量。

关于能量的损失：其他

管道“串联”：

  1. Q对于每一个管道都相等
  2. 列写伯努利方程的时候，最终的H是所有管道损失的和

管道“并联”：

  1. 起始点的$Q = Q_1+Q_2$
  2. 对于每个管子，都有:$h_{w1} = h_{w2}$
  3. 每个管子的流量存在一个比值关系：
    $$\frac{Q_1}{Q_2} = \sqrt{\frac{\lambda_2 \frac{l_2}{d_2}+(\Sigma K)_2}{\lambda_1 \frac{l_1}{d_1}+(\Sigma K)_1}}\frac{A_1}{A_2}$$

雷诺传输定律(RTT):

应用：改写的动量定理（不动的CV）

应用：改写的动量定理（CV相对会有移动）

首先复习下速度的合成：流体绝对的速度 = 流体相对于CV的流动速度 + CV的移动速度

$$\vec{V} = \vec{W} + \vec{V_{CV}}$$

对于我们所学习的课程，简化了很多，最后只剩下：

$$\sum \vec{F} = \int_{CS}\vec{W}\rho\vec{W}d\vec{A}$$

如果在CS上的分布是均匀的，能把积分变成一个简单的乘法,最后就是$\rho \vec{W} \vec {W} A$

应用：改写的角动量定理

$$\sum M_z(\vec{F}) = \frac{D\vec{H}}{Dt} = \frac{\partial}{\partial t} \int_{CV} \vec{r}\times\vec{V} \rho dV + \int_{CS} \vec{r}\times\vec{V}\rho \vec{V} d \vec{A}$$

说人话，1维、稳定的流动则可以简化许多：

$$\sum M_z = \dot{m}(\vec{r}{out}\times\vec{V}{out} - \vec{r}{in}\times\vec{V}{in})$$

注意 $\dot{m}$ 是质量流量。

扭矩T和做的功成正比，事实上，对于一根传动轴的功率，有：

$$P = \dot{W} = T_{torque}\omega$$

其中$\omega$事角速度。

计算质量流量的时候，注意要用到$V_{r}$，也就是径向的速度。

能量问题：

机械能

相信大家对于机械能的概念不再陌生了。在流体中，机械能由以下部分组成：

$$e_{mech} = \frac{P}{\rho} + \frac{V^2}{2} + gz$$

其中，第一项是流体的性质，压强除以密度；第二项是动能，速度的平方；第三项事势能，毕竟乘以了g和高度z。

如果不考虑损失，机械能的变化$\Delta e_{mech}$ 就是向流体做的功或者是流体向外界做功。

引入概念：效率。用符号$\eta$表示

$$\eta_{mech} = \frac{E_{mech,out}}{E_{mech,in}} = 1 - \frac{E_{mech,loss}}{E_{mech_in}}$$

写出了两种计算机械效率的公式。

在流体力学中，我们往往需要用类似的公式来计算一个比如涡轮对于流体的作用（改变了流体的压强、速度、高度这样）

$$\eta_{pump} = \frac{\Delta E_{mech,fluid}}{W_{shaft,in}}$$
上式的用来计算一个pump(水泵)的机械效率
$$\eta_{turbine} = \frac{W_{shaft,out}}{\Delta E_{mech,fluid}}$$
上式被用于计算涡轮的。

两式的区别就在于，水泵是机械做功，来移动液体的。涡轮一般是液体用自己的能量驱动机械。

热力学第一定律

用文字描述是：系统内部所储存的能量的时间变化率 = 传进来的热能 + 通过外界做功传进来的能量

$$\frac{D}{Dt}\int_{sys} e\rho dv = (Q_{in} - Q_{out})+(W_{in} - W_{out})$$

外界可能对流体做功的方式有三种，一是通过类似轴的方式来进行传递；二是流体的正应力做功；三是切应力做的功。

$$\dot{W_{normal stress}} = \int_{CS} -p \vec{V}d\vec{A}$$

其中p是压强。

知道了如何算流体的正应力做功之后，就可以改写咱的能量方程了：

当稳定流动的时候呢，是这样的：

$$\int_{CS} = (\mu + \frac{p}{\rho}+\frac{V^2}{2}+gz)\rho \vec{V} = \sum_{flow,out} \dot{m} (\mu+\frac{p}{\rho}+\frac{V^2}{2}+gz) - \sum_{flow,in} \dot{m} (\mu+\frac{p}{\rho}+\frac{V^2}{2}+gz)$$

但$\mu$ 是内能，不太好说怎么算。这里给出一种常用的方法是：内能和压力项组合起来，

$$\mu + \frac{p}{\rho} = h,h = C_pT$$

其中$h$是焓，是热力学中常用的一个量。这样之后进行改写：

$$\dot{m}[h_2 - h_1 + \frac{V_2^2 - V_1^2}{2} + g(z_2 - z_1)] = \dot{Q_{net_{in}}}+ \dot{W_{shaft_{netin}}}$$

将能量方程运用在非均匀流动中

当速度分布不均匀时，再计算速度平方项的时候往往需要用平均速度代替积分来进行估算。这就需要修正：(其中$\alpha$是修正系数，一般会给出)

$$\frac{p_{out}}{\rho}+\frac{\alpha_{out}\bar{V_{out}^2}}{2}+gz_{out} = \frac{p_{in}}{\rho}+\frac{\alpha_{in}\bar{V_{in}^2}}{2}+gz_{in}+w_{shaft_{netin}} - loss$$

除以g后，获得loss的“水头”形式：

$$\frac{p_{out}}{\gamma}+\frac{\alpha_{out}\bar{V_{out}^2}}{2g}+z_{out} = \frac{p_{in}}{\gamma}+\frac{\alpha_{in}\bar{V_{in}^2}}{2g}+z_{in}+\frac{w_{shaft_{netin}}}{g} - h_L$$

Loss归纳了热能、粘度这些的做功。

改写的伯努利方程

$$(\frac{p}{\rho g}+\frac{V^2}{2g}+z)_1 = (\frac{p}{\rho g}+\frac{V^2}{2g}+z)_2 + h_f - h_s $$

$h_f$ 为摩擦力的水头，$h_s$是水泵（外界做功）的水头

水头形式和实际功率之间仍然需要转换

$$P_{pump} = \rho g Q h_s$$
